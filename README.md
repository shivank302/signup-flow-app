# 🔐 Signup Flow App

A modern and responsive Signup/Login UI with a complete authentication flow using LocalStorage. This project simulates a real-world user authentication system with multiple screens and smooth transitions.

---

## 🚀 Live Demo

🔗 https://shivank302.github.io/signup-flow-app/

---

## 📌 Features

* 📝 User Signup with validation
* 🔑 Login system with stored credentials
* 🔄 Forgot Password functionality
* 🌙 Dark / Light mode toggle
* 💾 LocalStorage-based authentication
* 📱 Fully responsive design
* ✨ Smooth animations & transitions

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
